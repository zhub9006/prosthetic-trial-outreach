# Prosthetic Trial Outreach Project

Community outreach assessment for lower limb prosthetic trials, identifying enrollment gaps and evaluating **Shirley Ryan AbilityLab** (355 E Erie St, Chicago, IL 60611) as a community outreach hub for prosthetic users.

## Key Findings at a Glance

### Trial Gaps
| Metric | Finding |
|---|---|
| Total lower-limb prosthetic trials | 104 |
| Recruiting trials (estimated) | ~30 |
| Industry-sponsored recruiting trials | 33% |
| Federal/NI recruiting trials | ~3% (near zero) |
| Civic/nonprofit-funded trials | **0** |
| Phase 2–3 trials | **4** (tiny: 2–8 participants each) |
| Pediatric prosthetic trials | **0** |
| Rural trial sites | **0** |
| Non-English consent forms | **0** |

## Key Outreach Strategies (Top 5)

1. **Mobile DME Unit** — Monthly mobile prosthetist van for uninsured (~$15K/yr)
2. **Language Access** — Spanish consent forms + bilingual coordinators
3. **FQHC Partnerships** — South/West Side screening and referral
4. **Rural Tele-Outreach** — Virtual consult with VA rural sites
5. **Pediatric Trial** — Child-friendly prosthetic trial design
