# Prosthetic Trial Outreach

Community outreach assessment for lower limb prosthetic trials, identifying enrollment gaps and evaluating Shirley Ryan AbilityLab as a community outreach site.

## Key Findings

- 80% of prosthesis trials are Phase NA (device feasibility) — zero Phase 3+
- 72% academic-sponsored — single-site, small sample, K2 ambulators excluded from 85%+
- Zero trials in developing nations — Pakistan (1), Nigeria/Ghana (0), India (0)
- Demographic bias — elderly, minorities, women, rural, congenital deficiency underrepresented

## Shirley Ryan AbilityLab (355 E Erie St, Chicago)

- Walkability: 10/10 | Transit: 9.4/10 (48 CTA bus stops within 500m)
- Parking: 24/7 multi-storey 162m away (paid)
- CRITICAL GAP: Zero healthcare/pharmacy/restaurants within 500m

## Outreach Strategies
1. K2 Community Navigation Program
2. Multi-Site Referral Network + developing nation partnerships
3. Demographic Diversity Quotas (>=25% female, >=30% minority)
4. SRAL Location Leverage (rideshare, mobile clinic, garage rates)
5. Implementation Science (CBPR, Phase 2/3 hybrid, trial registry)
