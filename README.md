# Prosthetic Trial Outreach Project

Community outreach assessment for lower limb prosthetic trials, identifying enrollment gaps and evaluating **Shirley Ryan AbilityLab** (355 E Erie St, Chicago, IL 60611) as a community outreach hub for prosthetic users.

## Key Outreach Strategies

1. **Mobile DME Unit** — Monthly mobile prosthetist van for uninsured (~$15K/yr)
2. **Language Access** — Spanish consent forms + bilingual coordinators (~$8K/yr)
3. **FQHC Partnerships** — South/West Side screening and referral (~$5K/yr)
4. **Rural Tele-Outreach** — Virtual consult with VA rural sites (~$10K/yr)
5. **Pediatric Trial** — Child-friendly prosthetic trial design (~$20K/yr)
6. **Civic Funding Pipeline** — Philanthropic grants for equity-mandated trials (~$3K/yr)
