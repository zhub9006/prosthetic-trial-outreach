## Key Outreach Strategies (Top 5)

1. **Mobile DME Unit** — Monthly mobile prosthetist van for uninsured (~$15K/yr)
2. **Language Access** — Spanish, Polish, Arabic, Chinese consent forms + bilingual coordinators
3. **FQHC Partnerships** — South/West Side screening and referral pipeline
4. **Rural Tele-Outreach** — Virtual consult with VA rural sites; tele-rehabilitation
5. **Pediatric Trial Design** — Child-friendly prosthetic trial with elder-inclusive age ranges
