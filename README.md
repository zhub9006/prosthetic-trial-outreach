# Prosthetic Trial Outreach Project

Community outreach assessment for lower limb prosthetic trials, identifying enrollment gaps and evaluating Shirley Ryan AbilityLab as a community outreach site for prosthetic users.

**Repository:** https://github.com/zhub9006/prosthetic-trial-outreach  
**Last Updated:** July 2026

## Overview

This project assesses the current landscape of lower limb prosthetic clinical trials on ClinicalTrials.gov, identifies gaps in underserved population enrollment, and evaluates the Shirley Ryan AbilityLab (355 E Erie St, Chicago, IL 60611) as a potential community outreach hub.

## Key Findings Summary

### Clinical Trial Gaps (427 trials analyzed)
- **Industry sponsors** account for only ~19% of trials — severely limits tech access for underserved populations
- **96% of trials** are in "NA" or "Unknown" phase — no rigorous later-stage efficacy testing before market release
- **Rural US populations** are completely invisible — zero trials in the rural Midwest or Southern US
- **Diversity is rarely reported** — racial/ethnic demographics, female representation, and older adult enrollment are systematically absent

### Outreach Strategies
1. **Community-Based Screening Events** — mobile units at VA/FQHCs in rural areas
2. **Tele-Trial Participation** — hybrid model with travel stipends
3. **Industry-Academic Pipeline** — pro bono device donations + deployment to underserved sites
4. **Diversity Benchmarks** — mandatory demographic reporting (≥25% female, ≥30% minority)
5. **Patient Navigation & Transportation** — CTA reduced-fare cards, Divvy Adaptive, ride-share
6. **Older Adult Programming** — broader age ranges, caregiver stipends
7. **Rural Outreach via VA** — mobile prosthetics labs quarterly
8. **Data Transparency Dashboard** — open-source on GitHub
