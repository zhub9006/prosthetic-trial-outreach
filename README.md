## Key Outreach Strategies (Top 6)

1. **Mobile DME Unit** — Monthly mobile prosthetist van for uninsured (~$15K/yr)
2. **Language Access** — Spanish, Polish, Arabic, Chinese consent forms + bilingual coordinators (~$25K)
3. **FQHC Partnerships** — South/West Side screening and referral pipeline (~$40K/yr)
4. **Rural Tele-Outreach** — Virtual consult with VA rural sites; tele-rehabilitation (~$30K/yr)
5. **Pediatric Trial Design** — Child-friendly prosthetic trial with elder-inclusive age ranges (~$250K startup)
6. **SRAL On-Site Resource Corner** — DME samples, insurance navigation, walk-in adjustments (~$10K startup)
