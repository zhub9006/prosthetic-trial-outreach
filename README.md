# Prosthetic Trial Outreach Project

Community outreach assessment for lower-limb prosthetic trials, identifying enrollment gaps and evaluating **Shirley Ryan AbilityLab** (355 E Erie St, Chicago, IL 60611) as a community outreach hub for prosthetic users.

## Key Outreach Strategies

1. **Mobile DME Van** — Monthly mobile prosthetist van for uninsured (~$15K/yr)
2. **Language Access** — Spanish consent forms + bilingual coordinators (~$80K/yr)
3. **FQHC Partnerships** — South/West Side screening and referral
4. **Rural Tele-Outreach** — Virtual consult with VA rural sites (~$10K/yr)
5. **Pediatric Trial** — Child-friendly prosthetic trial design (~$100K+)
6. **Civic Sponsorship** — Community-based trial fund (~$50K startup)
7. **ADA Parking Support** — Negotiate spots + shuttle vouchers (~$12K/yr)
