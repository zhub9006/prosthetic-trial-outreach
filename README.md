## Key Findings at a Glance

### Trial Gaps (from ClinicalTrials.gov, 427 total lower-limb prosthetic studies)
| Metric | Finding |
|---|---|
| Total studies (clinicaltrials.gov) | **427** (514 with "prosthetic limb" term) |
| Industry-sponsored studies | 90 (17.5%) — commercial focus, no low-income targeting |
| Academic/university-sponsored | 367 (71.4%) — basic science, limited underserved recruitment |
| Federal/VA/NIH-funded studies | 46 (9%), **only 1 NIH-funded** — critical gap |
| Civic/nonprofit-funded trials | **0** |
| Phase 2–3 efficacy trials | **16 active (3.1%)** — severe pipeline gap |
| Pediatric prosthetic trials | **0** |
| Rural US trial sites | **0** |
| Non-English consent forms | **0** |
| Unknown phase/status trials | 119 (23.1%) — likely stalled recruitment |

### SRAL Site Accessibility (500 m radius)
| Category | Score | Key Finding |
|---|---|---|
| Transit | **10/10** | 10+ CTA bus stops, 6 Divvy bike stations, 24/7 access |
| Walkability | **10/10** | Flat, ADA sidewalks, excellent streetscape |
| Pharmacies | 4/10 | Walgreens ~200 m; no specialty prosthetic pharmacy |
| Parking | 4/10 | 4 paid garages; no ADA spots within 200m; low-clearance warnings |
| DME Vendors | **0/10** | **#1 site-level barrier for prosthetic users** |
| Parks/Social Services | 0/10 | None within 500 m |
| **Overall** | **5.4/10** | Strong transit/walkability; critical gaps in DME, parking, and social services |

## Key Outreach Strategies (Top 8)

1. **Mobile DME Unit** — Monthly mobile prosthetist van for uninsured (~$15K/yr)
2. **Language Access** — Spanish, Polish, Arabic, Chinese consent forms + bilingual coordinators (~$25K)
3. **FQHC Partnerships** — South/West Side screening and referral pipeline (~$40K/yr)
4. **Rural Tele-Outreach** — Virtual consult with VA rural sites; tele-rehabilitation (~$30K/yr)
5. **Pediatric Trial Design** — Child-friendly prosthetic trial with elder-inclusive age ranges (~$250K startup)
6. **SRAL On-Site Resource Corner** — DME samples, insurance navigation, walk-in adjustments (~$10K startup)
7. **Parking ADA Solutions** — Reserved ADA spots + shuttle service (~$5K)
8. **Adaptive Cycling Program** — Divvy adaptive cycles at nearby stations (~$10K startup)
